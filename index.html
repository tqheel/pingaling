Pingaling Is A Windows Command Line Tool For Pinging Websites

Technically, it does not use the ping protocol at all, but I liked the name (yes, it's a cheesy name, so sue me). Pingaling actually creates a set of http Requests for a given set of sites and then evaluates the http Responses.

How To Use

  • Open the project in Visual Studio (.Net 4.0).
  • Modify the included sites.xml file to include the sites that you want to "pingaling."
  • Build the project.
  • Copy the executable and sites.xml files from the debug folder to a directory of your choosing.
  • CD to that directory and type pingaling.
  • (Optional) Set-up Pingaling to run as as scheduled task in Windows Task Scheduler at the interval of your choosing.
  • Enjoy the goodness.
